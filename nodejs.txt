------------------------------------------------------------------
Cài đặt:
        cài nodejs
        node -v
        npm -v
        npm init                // tạo project
        npm install express
        tạo file index.js thư mục gốc:
            const express = require('express')
            const app = express()
            const port = 3000

            app.get('/', (req, res) => {
            res.send('Hello World!')
            })

            app.listen(port, () => {
            console.log(`Example app listening on port ${port}`)
            })
            node index.js           // sử dụng cli để chạy chương trình
View:
        Sử dụng Template:
            npm i express-handlebars
            src/index.js:
                const handlebars = require('express-handlebars')
                const path = require('path');
                app.engine('hbs', handlebars.engine({extname: '.hbs'}));        // đổi đuổi file với extname
                app.set('view engine', 'hbs');
                app.set('views', path.join(__dirname, 'resources/views'));
                app.get('/hello3', (req, res) => {
                    res.render('home')
                })
            src/resources/views/layouts/main.handlebars:
                <head>
                </head>
                <body>
                    {{> header }}
                    {{{ body }}}
                    {{> footer }}
                </body>
            src/resources/views/home.handlebars: <h1>hello</h1>
            src/resources/views/partials/header.handlebars: <h1>header</h1>
            src/resources/views/partials/footer.handlebars: <h1>footer</h1>

Services:
        Đổi thư mục lưu file index.js:
            scr/index.js
            package.json->main:
        Thêm file static:
            src/public/img/logo.png:
            src/index.js:
                app.use(express.static(path.join(__dirname, 'public')))
            http://localhost:3000/img/logo.png
Config:
        tự động update code khi chạy chương trình:
            npm i nodemon --save-dev
            package.json->script->start: "nodemon index.js"
            npm start
        Debug:
            npm i nodemon --save-dev
            package.json->script->start: "nodemon --inspect index.js"
            vào trình duyệt => inspect => biểu tượng node => sources => index.js => break point 
        logger:
            npm i morgan --save-dev             // logger xuất hiện trên command line
            var morgan= require('morgan')       // sử dụng trên file index.js hoặc file chạy chương trình
            app.use(morgan('combined'))
        biên dịch sass:
            npm i node-sass --save-dev
            src/public/resources/scss/app.scss
            src/public/css/app.css
            package.json->script:
                "watch": "node-sass --watch src/resources/scss/ --output src/public/css/" // tham số watch để theo dõi thay đổi;  các tham số lần lượt là input output biên dịch file scss sang css
            npm run watch
Các khái niệm:
        package.json->dependencies          // quản lý các thư viện
        package.lock.json                   // quản lý các thư viện của thư viện
        --save-dev                          // chỉ cài thư viện trên project; sẽ xuất hiện ở devDependencies
        biến __dirname                      // đường dẫn đầy đủ đến thư mục gốc dự án

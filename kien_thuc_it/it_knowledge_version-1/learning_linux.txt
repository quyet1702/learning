các câu lệnh linux cơ bản:
	pwd: Hiển thị đường dẫn thư mục làm việc hiện tại.
	ls: Liệt kê tất cả các tệp tin và thư mục trong thư mục hiện tại.
	cd: Di chuyển đến một thư mục khác.
	mkdir: Tạo mới một thư mục.
	rm: Xóa tệp tin hoặc thư mục.
	cp: Sao chép tệp tin hoặc thư mục.
	mv: Di chuyển hoặc đổi tên tệp tin/thư mục.
	cat: Hiển thị nội dung của một tệp tin.
	grep: Tìm kiếm các chuỗi trong tệp tin hoặc đầu ra của lệnh khác.
	chmod: Thay đổi quyền truy cập của tệp tin hoặc thư mục.
	chown: Thay đổi chủ sở hữu của tệp tin hoặc thư mục.
	ps: Hiển thị các tiến trình đang chạy.
	top: Hiển thị danh sách các tiến trình đang chạy và sử dụng tài nguyên hệ thống.
	ifconfig: Hiển thị thông tin về các giao diện mạng.
	ssh: Kết nối với một máy chủ từ xa qua giao thức SSH.

redis:
	sudo apt install lsb-release curl gpg
	curl -fsSL https://packages.redis.io/gpg | sudo gpg --dearmor -o /usr/share/keyrings/redis-archive-keyring.gpg
	echo "deb [signed-by=/usr/share/keyrings/redis-archive-keyring.gpg] https://packages.redis.io/deb $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/redis.list
	sudo apt-get install redis
	redis-server				// khởi động redis
	redis-cli				// kết nối redis
	ping

Cài đặt rabbitmq:
	sudo apt update
	sudo apt install curl software-properties-common apt-transport-https lsb-release
	curl -1sLf 'https://dl.cloudsmith.io/public/rabbitmq/rabbitmq-erlang/setup.deb.sh' | sudo -E bash
	sudo apt update
	sudo apt install erlang
	sudo apt install -y rabbitmq-server
	sudo service rabbitmq-server status
	sudo service rabbitmq-server stop
	sudo service rabbitmq-server start
	sudo service rabbitmq-server restart
	sudo rabbitmqctl add_user admin pwd123			// sét user
	sudo rabbitmqctl set_user_tags admin administrator	// set quyền
	sudo rabbitmqctl set_permissions -p / admin ".*" ".*" ".*"	// set quyền cho nhiều user
	hostname -I					// truy cập vào port mặc định 15672

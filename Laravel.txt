Cách hoạt động của Chương Trình Laravel:
	- Chạy chương trình:
		+ File khởi động:
			- index.php
		+ Tạo instance của ứng dụng:
			- bootstrap/app.php
		+ Tải các cấu hình (config):
			- gồm các file config cần thiết và các file config chạy thủ công qua command
			- config/app.php
			- config/database.php
			- config/services
			- config/auth.php
		+ Khởi động các dịch vụ (services):
			+ Các Service Providers
	- Xử lý các request:
		System => (Request => Middleware => Validate => Controller(query => Model  => response) => view)

		-> chỉnh sửa thông tin user (route/web -> ProfileController.update -> App\Http\Controllers\ProfileController)
		-> xóa user (route/web -> ProfileController.destroy -> App\Http\Controllers\ProfileController)
		-> xác nhận mật khẩu (route\auth\confirm-password -> ConfirmablePasswordController.show -> view -> ConfirmablePasswordController.store)
		-> đăng xuất (route\auth\logout -> AuthenticatedSessionController.destroy -> view)

				public function boot()
